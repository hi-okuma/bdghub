ボドゲハブ(仮) : 0001.ゲームについて

Created by 大熊浩也, last modified on 5月 05, 2025

ゲーム概要

発言したら脱落するNGワード。本人だけには分からないNGワードを、全員が持っています。他の人のNGワードを引き出しつつ、自分のワードを推測し、回避しましょう。

参考リンク）
https://dokopa.com/ngwordgame/

機能

MVP

  • タイトル画面

    • プレイヤー一覧

    • 準備完了ボタン

    • ゲーム終了ボタン（ホストプレイヤーのみ表示）

  • ゲーム画面

    • プレイヤー一覧

      • ニックネームとNGワード（１つ）の表示

      • 脱落表示

    • 自分が脱落したことを申告するボタン

  • 結果発表画面

    • プレイヤー一覧

    • もう一度遊ぶボタン

    • タイトルに戻るボタン

バックログ

  • 過去のゲーム（当日中）で割り当てられたNGワードは出ないようにする

  • 1プレイヤーに割り当てられるNGワード数を変更できる

  • 複数プレイヤーに同じNGワードが割り当てられる設定のON/OFF
